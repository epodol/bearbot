# Bear Bot

## Setting up your workspace

1. Clone the repo
   ```
   > git clone https://github.com/epodol/bearbot.git
   > cd bearbot
   ```
1. Install dependencies
   ```
   > npm install
   ```
1. Duplicate `config [example]`, rename it `config`, and fill in the information
1. Start Bear Bot
   ```
   > npm run start
   ```
